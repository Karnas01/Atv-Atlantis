##  Como rodar o projeto
- Crie uma nova pasta, acesse o CMD na nova pasta e digite o seguinte comando:
  
```
  git clone https://github.com/Karnas01/Atv-Atlantis.git .
```
- Acesse a branch AtvIV:

```
  git checkout AtvIV
```
- Instale as dependências:
  
```
  npm i
```
- Inicie a aplicação:
```
  npm start
```
